# Sheet Generator

Monitoring & Reference Sheet Generator

## Installation & Setup

1. **Install dependencies**:

   ```bash
   cd client && npm install
   cd ../server && npm install
   cd ..
   ```

2. **Install root dev dependencies**:

   ```bash
   npm install
   ```

3. **Start development servers**:

   ```bash
   npm run dev
   ```
